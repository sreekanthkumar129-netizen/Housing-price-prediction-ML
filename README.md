# Housing Price Prediction using Linear Regression

##  Overview
This project applies **linear regression** to predict housing prices. The workflow includes data preprocessing, exploratory data analysis (EDA), model training, and evaluation. It demonstrates how regression models can be used in real estate price forecasting.

---

##  Steps Implemented
1. **Data Loading & Exploration**
   - Imported dataset into Pandas DataFrame
   - Checked dataset shape, column types, and missing values
   - Reviewed statistical summaries and distributions

2. **Data Preprocessing**
   - Handled missing values
   - Normalized/standardized features where necessary
   - Selected relevant features (e.g., square footage, number of rooms, location indicators)

3. **Exploratory Data Analysis (EDA)**
   - Visualized trends and correlations using Matplotlib & Seaborn
   - Plotted housing price distributions
   - Checked feature relationships with scatter plots and heatmaps

4. **Model Building**
   - Implemented **Linear Regression** using scikit-learn
   - Split dataset into training and testing sets
   - Trained regression model on training data

5. **Model Evaluation**
   - Predicted housing prices on test data
   - Calculated performance metrics (MAE, MSE, RMSE, R²)
   - Visualized predicted vs actual values

6. **Results & Insights**
   - Observed accuracy of predictions
   - Highlighted limitations and potential improvements
